# /synthetische-data — Synthetische seed-datasets

Deze map bevat uitsluitend **synthetische** (verzonnen) testdata voor ontwikkeling
en test. Deze data voedt later de Laravel-seeders.

## Harde regel (AVG)

- **NOOIT echte persoonsgegevens** in deze map, in geen enkele fase.
- Geen productiedata uit de oude Access-database (`IUTSTD-*.mdb`).
- BSN alleen na expliciet akkoord en dan nog uitsluitend synthetisch gegenereerd.

## Inhoud (komt in latere fase)

- Synthetische studenten, opleidingen, inschrijvingen, toetsonderdelen en
  resultaatregels.
- Generatiescript/documentatie voor het reproduceerbaar opbouwen van de dataset.

Nog leeg — de synthetische seed-dataset wordt in een latere sessie gegenereerd
(afronding Fase 0).
