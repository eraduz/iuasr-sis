# /docs — Ontwerp- en projectdocumentatie

Deze map bevat de ontwerpdocumentatie van het managementsysteem: functioneel ontwerp (FO),
technisch ontwerp (TO), datamodel, rolmodel, autorisatiematrix en AVG-stukken
(o.a. DPIA-opzet).

## Inhoud (opbouw per fase)

- Functioneel Ontwerp (FO) — Fase 1
- Technisch Ontwerp (TO) — Fase 2
- Datamodel en rolmodel
- Autorisatiematrix (rolscheiding SZ / Docent / Examencommissie / Directie / Admin)
- AVG: DPIA-opzet en verwerkingsregister

## Regels

- Taal: Nederlands (U-vorm), geen emoji.
- Geen echte persoonsgegevens in voorbeelden — uitsluitend synthetische data.
- Brondocumenten `IUASR-Plan-van-Aanpak.docx` en `Database programma
  omschrijving.docx` staan (voorlopig) in de repo-root.
